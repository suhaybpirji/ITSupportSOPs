# Reset a User Password (Active Directory)

## Purpose
To reset a user’s password in Active Directory when they forget it or are locked out.

## Steps
1. Open Active Directory Users and Computers.
2. Locate the user.
3. Right-click → Reset Password.
4. Enter new password.
5. Tick “User must change password at next logon”.
6. Click OK.

## Verification
Ask the user to log in again.
