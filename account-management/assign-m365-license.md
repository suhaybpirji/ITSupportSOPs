# Assign a Microsoft 365 License to a User

## Purpose
To grant a user access to Microsoft 365 services such as Outlook, Teams, OneDrive, and SharePoint.

## Steps
1. Open the Microsoft 365 Admin Center.
2. Go to Users → Active Users.
3. Search for and select the user.
4. Click on Licenses and Apps.
5. Select the appropriate license (e.g., Business Premium, E3, E5).
6. Enable or disable specific apps as required.
7. Click Save Changes.

## Verification
- Ask the user to sign in at https://portal.office.com.
- Check if the mailbox has been created (may take 5–10 minutes).
- Confirm Teams and OneDrive access.

## Notes
Ensure the organisation has available license seats before assigning.
