# Create a New User in Active Directory

## Purpose
To onboard a new employee by creating their Active Directory account with the correct details and permissions.

## Steps
1. Open Active Directory Users and Computers.
2. Navigate to the correct OU (e.g., Users or Department OU).
3. Right-click → New → User.
4. Enter:
   - First name
   - Last name
   - Username (SAMAccountName)
5. Set a temporary password.
6. Tick “User must change password at next logon”.
7. Add the user to required groups:
   - Department groups
   - Security groups
   - Distribution lists
8. Assign Microsoft 365 license if required.

## Verification
- Confirm the user appears in the correct OU.
- Check group membership.
- Test login if possible.

## Notes
Follow company naming conventions and document the account creation in the ticketing system.
