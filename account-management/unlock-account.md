# Unlock a User Account (Active Directory)

## Purpose
To unlock a user account that has been locked due to multiple failed login attempts.

## Steps
1. Open Active Directory Users and Computers.
2. Search for the user account.
3. Right-click the user → Properties.
4. Go to the Account tab.
5. Tick “Unlock account” if the option is visible.
6. Click Apply → OK.

## Verification
Ask the user to try logging in again.

## Notes
If the user gets locked repeatedly, check:
- Mobile devices with old passwords
- Outlook cached credentials
- Mapped drives using old credentials
