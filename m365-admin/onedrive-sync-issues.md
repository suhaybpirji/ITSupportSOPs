# Fix OneDrive Sync Issues (Microsoft 365)

## Purpose
To resolve problems where OneDrive fails to sync files, shows errors, or stops updating changes between the local device and the cloud.

## Symptoms
- OneDrive not syncing
- Red ❌ or yellow ⚠️ icons on files
- “Sync pending”
- “Processing changes”
- Files stuck uploading
- OneDrive not starting
- Duplicate files (filename‑PC, filename‑conflict)
- Shared libraries not syncing

## Steps

### 1. Check OneDrive Status
Click the OneDrive icon in the taskbar.

Look for:
- ✔️ Up to date  
- 🔄 Syncing  
- ❌ Error  

If the icon is missing → OneDrive not running.

### 2. Restart OneDrive
1. Right‑click OneDrive icon → Quit OneDrive
2. Reopen from Start menu:
   OneDrive

### 3. Check Internet / VPN
OneDrive may fail if:
- VPN blocks Microsoft endpoints
- Firewall restricts traffic
- Network is unstable

Try:
- Disconnect VPN
- Switch network
- Restart router (home users)

### 4. Check Storage Space
Ensure:
- Local disk has free space
- OneDrive cloud storage is not full

Check cloud storage:
https://onedrive.com
