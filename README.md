## 🎯 Purpose of This Repository

This project demonstrates:

- Real-world troubleshooting workflows  
- Clear, structured IT documentation  
- Active Directory & M365 knowledge  
- Windows Server & client support  
- Networking fundamentals  
- Security best practices  
- ITIL-style processes  

Perfect for showcasing 2nd Line readiness.

---

## 🧩 Categories Included

### 🔐 Account Management  
Password resets, unlocking accounts, onboarding/offboarding, M365 licensing.

### 🗂 Active Directory  
Domain joins, GPO issues, replication checks, lockout investigations.

### 🌐 Networking  
DNS, VPN, Wi-Fi, printers, mapped drives.

### 🖥 Windows Client  
Outlook, Teams, updates, performance issues.

### 🏢 Windows Server  
Services, event logs, RDP, disk space, DNS/DHCP.

### ☁️ M365 & Intune  
MFA resets, device wipes, mailbox checks, shared mailbox access.

### 🔒 Security  
BitLocker, antivirus, phishing, password policies.
